# Subtractor
Use NN model to implement Arithmetics operations. 
It can be used to calculate **three-digit numbers addition and substraction**.

## Execute
`python main.py`

## Requirements
`pip install -r requirements.txt`
* pandas==0.22.0
* numpy==1.14.0
* seaborn==0.8.1
* scikit-learn==0.19.1
* keras==2.1.6
